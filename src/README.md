# Индексатор папок

Консольная утилита на Python для индексации файлов с сохранением данных в SQLite.

## Запуск

1. Открыть проект в PyCharm
2. Настроить Run Configuration: указать `src/main.py` как точку входа
3. В поле Parameters указать путь к папке для сканирования

Или через терминал:

python src/main.py C:\Users\Мат\Desktop\Практика

## Структура проекта

src/

main.py        — точка входа, принимает путь к папке

db.py          — инициализация базы данных SQLite

student_meta.md — информация о студенте

journal.md     — журнал работ

data/

app.db         — база данных (создаётся автоматически)
